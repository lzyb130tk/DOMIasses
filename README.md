# Asset Pro v5 (单文件资产管理)

一个极简、现代化的个人资产管理工具，完全运行在浏览器端，数据存储在本地。

## ✨ 特性

- **单文件设计**：核心逻辑与样式全部封装在一个 `.html` 文件中，随处可用。
- **数据隐私**：所有数据存储在浏览器 IndexedDB 中，不上传任何服务器。
- **现代化 UI**：采用 iOS 设计语言，支持自适应布局。
- **图表统计**：自动计算总资产、分类统计。
- **时间轴视图**：以时间流形式回顾你的败家（资产积累）历史。
- **数据备份**：
  - 支持导出 ZIP 备份（包含图片和 JSON 数据）。
  - 支持从 ZIP 完整恢复。
- **PWA 支持**：可添加到手机主屏幕，像原生 App 一样使用。

## 📱 如何使用

### iOS (iPhone/iPad)

由于 iOS 系统限制，无法直接安装 APK。

1. 在 **Safari** 浏览器中打开 `isets.html` 文件（或将其部署到静态网页托管）。
2. 点击底部工具栏的 **分享** 按钮。
3. 选择 **"添加到主屏幕"**。
4. 现在你可以像原生 App 一样从桌面启动它，且全屏运行。

### Android

1. **直接使用**：使用 Chrome 打开，点击菜单中的 "安装应用" 或 "添加到主屏幕"。
2. **打包为 APK**：使用 [Demo2APK](https://github.com/DeadWaveWave/demo2apk) 将此 HTML 文件打包成 Android APK 安装包。

### Windows / Mac

直接双击 `isets.html` 使用 Chrome / Edge 浏览器打开即可。

## 💾 数据备份与迁移

**极其重要**：因为数据存储在浏览器缓存中，浏览器的"清除缓存"功能可能会导致数据丢失。

- 请务必**定期点击设置页面的备份数据**按钮。
- 生成的 ZIP文件包含了你的所有资产信息和图片。
- 更换设备时，只需在新设备上点击"恢复数据"并选择 ZIP 文件即可。
